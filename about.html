About Communote

Communote offers a web-based open source communication platform for enterprises, institutions, NGOs and communities. The Communote project was initiated by Communardo, a leading social business and collaboration solution provider.

Download and Contribute

Communote is an open source project. The code is licensed under the Apache License, Version 2.0.

Try and tell

Download and try Communote. Feel free to let the team know your feature suggestions and bug reports. Your participation in the further development of the software will be highly welcomed by the user community.

Test and file bug reports

Help testing new Communote releases on different platforms and databases and create good bug reports.

Develop plugins

Develop cool plugins and share them with other Communote users.

Ask questions

Let us know about your problems with plugin development or answer questions of other developers (link zu dem SaaS Client).

Fix bugs

Help fixing bugs in Communote's core and implement new core features or extension points for plugins. You may also edit the documentation to contribute own code and to deepen topics.
